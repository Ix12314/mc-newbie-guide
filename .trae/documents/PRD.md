## 1. 产品概述
一个面向技术爱好者的实时新闻聚合网站，以编辑级杂志气质呈现来自 Hacker News 的实时数据流，提供两个核心页面：**新闻**（实时头条列表）和**更多**（热门 / Best / Show HN / 招聘等多源数据视图）。
- 解决用户在碎片时间快速获取高质量技术圈实时资讯的需求
- 目标用户为开发者、产品经理、科技爱好者；价值在于"一个页面看懂技术圈正在发生什么"

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 匿名访客 | 无需注册 | 浏览全部实时新闻、搜索、分类筛选、跳转原文 |

### 2.2 功能模块
1. **新闻页**：实时头条列表、关键词搜索、分类筛选（Top / New / Best）、自动刷新、加载更多、跳转原文与评论
2. **更多页**：多 Tab 数据视图（热门 / Show HN / 招聘）、实时统计仪表盘、数据来源说明、回到顶部

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 新闻 | 顶部状态栏 | 显示最后刷新时间、数据源标识、手动刷新按钮 |
| 新闻 | 搜索栏 | 关键词实时过滤标题（前端过滤当前批次） |
| 新闻 | 分类切换 | Top / New / Best 三种数据源切换 |
| 新闻 | 新闻卡片 | 标题、来源域名、得分、作者、相对时间、评论数、原文链接、评论链接 |
| 新闻 | 加载更多 | 增量加载下 20 条，保持滚动位置 |
| 新闻 | 骨架屏 | 数据加载中显示卡片占位动画 |
| 更多 | Tab 视图 | 热门（Top）/ Show HN / 招聘（Job）三种实时数据 |
| 更多 | 统计仪表盘 | 当前批次最高分、平均分、评论总数、作者数 |
| 更多 | 数据源说明 | 展示 API 来源、更新机制、隐私声明 |
| 更多 | 回到顶部 | 浮动按钮长列表快速返回 |

## 3. 核心流程
用户进入新闻页 → 自动拉取 Top Stories 实时数据 → 浏览/搜索/切换分类 → 点击卡片跳转原文或评论 → 切换到更多页查看 Show HN/招聘/统计 → 手动刷新获取最新数据

```mermaid
    "自动拉取 Top Stories" --> "渲染卡片列表"
    "渲染卡片列表" --> "用户搜索或切换分类"
    "用户搜索或切换分类" --> "重新拉取对应数据源"
    "重新拉取对应数据源" --> "点击卡片"
    "点击卡片" --> "跳转原文或评论"
    "跳转原文或评论" --> "切换到更多页"
    "切换到更多页" --> "查看 Show HN / 招聘 / 统计"
    "查看 Show HN / 招聘 / 统计" --> "手动刷新"
    "手动刷新" --> "自动拉取 Top Stories"
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：纯白底 (#FFFFFF) + 墨黑 (#0A0A0A) + 中性灰阶 (#F5F5F5 / #A1A1AA) + 强调色琥珀 (#F59E0B) 与翡翠 (#10B981)
- **按钮风格**：极简圆角（rounded-full），主操作墨黑实底白字，次操作描边玻璃质感
- **字体**：标题用 "Fraunces"（衬线杂志感）+ 正文用 "Inter" / 中文回退 "PingFang SC"，标题与正文形成强对比
- **布局风格**：编辑级杂志排版，强网格、大留白、左对齐大标题、分隔线细而锐利
- **图标风格**：lucide-react 线性图标，统一 1.5px 描边，圆角端点

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 新闻 | 顶部状态栏 | 顶部细条，左：实时数据源标签 + 最后刷新时间，右：刷新按钮带旋转动画 |
| 新闻 | 大标题区 | 大号衬线 "实时新闻" + 副标题 + 分类切换胶囊 |
| 新闻 | 搜索栏 | 圆角输入框，左侧搜索图标，聚焦时描边变色 |
| 新闻 | 新闻卡片 | 序号（大号衬线）+ 标题 + 元信息行 + 评论/原文链接，hover 微微上浮 |
| 新闻 | 加载更多 | 居中描边按钮，加载时显示 spinner |
| 更多 | Tab 导航 | 顶部胶囊式 Tab，激活态墨黑底白字 |
| 更多 | 统计卡片 | 4 列网格，大号数字 + 小号说明，琥珀/翡翠点缀 |
| 更多 | 数据源说明 | 卡片式说明，列出 API 端点与刷新策略 |

### 4.3 响应式
- 桌面优先设计（1280px+ 完整双栏体验）
- 平板自适应（768-1280px 调整列数与字号）
- 移动端适配（<768px 单列布局，触摸优化，导航胶囊可横向滚动）

### 4.4 视觉氛围
- 大留白 + 细分隔线营造编辑级秩序感
- 序号大号衬线字体作为视觉锚点
- 卡片 hover 时极轻微上浮 + 阴影增强
- 加载状态使用骨架屏而非 spinner 占据主区域
- 配色克制，强调色仅用于关键指标和激活态
